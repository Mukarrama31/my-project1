Basic Commands

> git config --list
    List config settings

> git init
    create a new git Repository

> git status
    Show the current branch and new/Modified files in your Repository

> git add <file> 
> git add . 
    Add one or all files to the staging area. Changes to be committed

> git rm --cached <file>: remove a file from an staging area

> git diff <file> 
    Track the difference between the changes made on a file 

> git diff --color-words <file>
    Show the difference in a clean and simple way

> git commit -m 'descriptive message'
    create a snapshot(save point) in your project.
